# Yulu Demand Analysis & Pricing Insights

## Project Overview
This project analyzes rental data from Yulu to understand demand patterns and identify key factors influencing bike rentals.

The insights derived aim to support pricing strategy, fleet optimization, and operational planning.

## Business Objective
- Analyze demand trends across time and seasons
- Identify factors impacting rentals
- Support data-driven pricing decisions
- Improve operational efficiency

## Dataset
The dataset includes:
- Date & time
- Season
- Weather
- Working day
- User type
- Total rentals

## Tools Used
- Python
- Pandas, NumPy
- Matplotlib, Seaborn
- SciPy

## Key Insights
- Demand peaks during working days and peak hours
- Weather significantly impacts rentals
- Registered users drive consistent demand

## Business Recommendations
- Implement dynamic pricing during peak hours
- Increase fleet availability during high-demand seasons
- Introduce promotions during low-demand periods

## Conclusion
This analysis highlights how demand patterns can be leveraged to optimize pricing and operations in micro-mobility services.
